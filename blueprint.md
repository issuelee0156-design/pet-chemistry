
# Blueprint: Myeongli-based Applications

## Overview

This project provides a suite of web applications based on Myeongli-hak (Asian astrology). The initial applications include a Lotto Number Generator and a Pet Compatibility Tester. The applications are designed to be interactive, visually appealing, and provide a unique user experience.

## Project Structure & Design

### Style & Design
- **Theme:** Modern, clean, and intuitive design.
- **Layout:** Responsive layout that works on both mobile and web.
- **Color Palette:** A vibrant and energetic color palette is used to create a visually appealing interface.
- **Typography:** Expressive typography is used to emphasize key information and improve readability.
- **Iconography:** Icons are used to enhance user understanding and navigation.
- **Interactivity:** Interactive elements have a "glow" effect to provide visual feedback to the user.

### Features
- **Lotto Number Generator:** Generates lottery numbers based on the user's name, birth date, and a selected date.
- **Pet Compatibility Test:** Analyzes the compatibility between a pet and its owner based on a photo.
- **Internationalization (i18n):** The applications support both Korean and English languages.
